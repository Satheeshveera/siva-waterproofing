## 🚀 Setup Instructions

### Step 1: Formspree Email Integration (FREE)

1. Go to https://formspree.io
2. Click "Sign Up" and create a free account
3. Click "New Project" and create a project
4. Add your email: sivasatheesh05@gmail.com
5. You'll get a Form ID (e.g., f_xyz123abc)
6. Copy the Form ID
7. Open `index.html` in the contact form section:
   ```html
   <form id="contactForm" method="POST" action="https://formspree.io/f/YOUR_FORM_ID">
   ```
8. Replace `YOUR_FORM_ID` with your actual ID from Formspree
9. Save and push to GitHub

**That's it! Now all form submissions will be emailed to you.**

### Step 2: WhatsApp Integration (Already Done)

✅ WhatsApp floating button is already integrated!
- Click the green WhatsApp button to chat
- Service-specific messages are pre-filled
- No setup needed - it's ready to use!

### Step 3: Google Analytics (Optional)

1. Go to https://analytics.google.com
2. Create a new property
3. Get your Measurement ID (G-XXXXXXXXXX)
4. In `index.html`, replace `G-XXXXXXXXXX` with your ID
5. Save and push

### Step 4: Facebook Pixel (Optional)

1. Go to https://business.facebook.com/
2. Create a pixel
3. Get your Pixel ID
4. In `index.html`, replace `1234567890` with your ID
5. Save and push
