### Update Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #0d6efd;
    --secondary-color: #00d4ff;
    --dark-color: #1a1a1a;
    --light-color: #f8f9fa;
}
```

### Add Images
Replace placeholder divs with actual images:
```html
<img src="path/to/image.jpg" alt="Description">
```
